# TEC-1 ROMs

Bundled TEC-1 monitor ROMs for Debug80.

## MON-1B

- File: `mon-1b.hex`
- Source: `/Users/johnhardy/Documents/projects/Software/monitors/Mon-1/Mon-1B/mon1B.asm`
- License: GPL-3.0

If you want to use a different ROM (MON-1A, MON-2, JMON), set `tec1.romHex`
in your project config to point at your chosen ROM file.
