# Support

Use the GitHub issue tracker for bugs, regressions, and feature requests:

https://github.com/jhlagado/debug80/issues

For debugging issues, include:

- Debug80 version.
- VS Code version and operating system.
- Platform profile (`simple`, `tec1`, or `tec1g`).
- Relevant `debug80.json` target.
- Source file path used as the program entry point.
- Any build/debug output from the VS Code Debug Console.
- Whether the issue appears in a packaged VSIX build or only in the Extension
  Development Host.

For source-mapping or breakpoint problems, also include the generated D8 debug
map when possible.
